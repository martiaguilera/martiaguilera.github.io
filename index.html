Simple Web Server (SWS)

Simple Web Server (SWS) es una herramienta liviana que te permite convertir cualquier carpeta en un servidor web local en cuestión de segundos. Diseñada para desarrolladores y estudiantes, ofrece una interfaz gráfica para seleccionar carpetas y una consola en tiempo real para monitorear el estado del servidor.

  • Servidor web local en segundos
  • Compatible con archivos estáticos (HTML, CSS, JS)
  • Interfaz gráfica intuitiva
  • Consola integrada para monitoreo
  • Sin dependencias adicionales
